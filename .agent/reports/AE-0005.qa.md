## Fix Validated

**File**: `router.py:17` — `strategies_router` included before `crud_router`

**Root cause**: `strategies_router` was not included in the parent router at HEAD. `GET /api/carousels/strategies` fell through to `crud_router`'s `GET /{project_id}`, where `"strategies"` failed UUID parsing → 422.

**Fix**: Added import and `include_router(strategies_router, prefix=PREFIX)` before `crud_router`. The literal path `/strategies` is now registered before the dynamic path `/{project_id}`, so FastAPI matches the correct handler.
