### Summary
`gates.sh` now distinguishes "devDependency not installed" from a real violation. A new
sourceable preflight `scripts/lib/require_tool.sh::require_tool` probes
`$FRONTEND_BIN_DIR/<tool>` (npm-local bin, not `which`/`npx`) and returns `EXIT_SKIP` (77)
with an actionable "run `cd frontend && npm ci`" message on a miss. The four jscpd/knip
gates (`duplication`, `dead_code`, `duplication_tests`, `dead_files`) call it first, so a
missing tool reports SKIP locally (and FAIL under `GATES_REQUIRE_ALL=1` in CI) instead of
a raw exit-127 FAIL or a swallowed false PASS.
