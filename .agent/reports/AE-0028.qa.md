## 🟠 Warning Findings (Unchanged)

### 6. Exception Details Leaked in Admin Responses
- **File:** `backend/src/rag_backend/api/routes/carousels/admin.py` (lines 71–73, 111–114)
- **Issue:** `refresh_design_tokens` and `render_missing_slides` catch bare `Exception` and append `f"{project.id}: {exc}"` to the JSON response, leaking internal exception details.
- **Action:** Replace with generic error messages and log server-side with `logger.exception(...)`.
- **Reference:** OWASP A10 — Mishandling of Exceptional Conditions

### 7. Wildcard CORS Default with Credentials
- **File:** `backend/src/rag_backend/infrastructure/config/settings.py` (line 23)
- **File:** `backend/src/rag_backend/api/app.py` (lines 177–189)
- **Issue:** `allowed_origins` defaults to `"*"`. Combined with `allow_credentials=True`, this is a security misconfiguration.
- **Action:** Change default to empty string or safe placeholder; require explicit configuration in production.
- **Reference:** OWASP A02 — Security Misconfiguration

### 8. FastAPI Path Validation Missing on Media Filename
- **File:** `backend/src/rag_backend/api/routes/carousels/media.py` (lines 231, 261)
- **Issue:** `get_carousel_image` and `get_carousel_slide_image` accept `filename: str` without `FastPath(pattern=...)` constraint.
- **Action:** Add `FastPath(pattern=r"^[a-zA-Z0-9_\-\.]+$")` to route parameters.
- **Reference:** OWASP A05 — Injection

### 9. QA Script Does Not Enforce HTTPS
- **File:** `scripts/carousel_visual_qa.py` (lines 170–173)
- **Issue:** `_assert_http_url` validates `http` or `https` but does not reject `http`. Credentials are sent to `/api/auth/token`.
- **Action:** Reject `http` scheme and require `https` when `base_url` is not `localhost`/`127.0.0.1`.
- **Reference:** OWASP A04 — Cryptographic Failures

### 10. RenderLocale Nested Function in React Component
- **File:** `frontend/src/app/dashboard/create/workspace/phase-review/content-phase-review.tsx` (lines 147–274)
- **Issue:** `renderLocale` is defined as a nested function inside a `map` callback. Recreates on every render, prevents memoization.
- **Action:** Extract to a separate component.
- **Reference:** React anti-pattern — function re-creation

### 11. Defensive NotImplementedError Catches in image_generation_records.py
- **File:** `backend/src/rag_backend/application/services/carousel/image_generation_records.py` (lines 48, 72)
- **Issue:** Catches `AttributeError` and `NotImplementedError` when calling repository methods. If the protocol is required to implement these, the catches silently mask missing persistence.
- **Action:** Confirm if `CarouselRepository` is required to implement these. If yes, remove the catches and enforce the contract.
