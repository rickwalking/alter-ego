### Summary

(interrupt) instead of routing to `END`, so a final-review send-back reliably
re-enters the targeted phase and regenerates it while reusing images. `get_state`

carousel stays publishable; any revise drops the publish approval in graph state

re-open carousels that were already terminated (approved) before deploy.
