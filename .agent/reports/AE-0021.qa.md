## Edge Cases
- **Missing API key**: Returns actionable `RuntimeError` with `_MSG_MISSING_KEY`
- **403 Forbidden (org verification)**: Returns actionable `RuntimeError` with `_ERR_NOT_VERIFIED_DETAIL`
- **Empty response data**: Returns `_ERR_NO_RESPONSE_DATA`
- **Missing b64_json payload**: Returns `_ERR_MISSING_B64_JSON`
- **API connection failure**: Caught as `APIError`, wrapped with structured detail
- **Partial error response**: Gracefully handles missing `param` field
- **Non-JSON error body**: Falls back to string representation
