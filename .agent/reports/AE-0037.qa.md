## Edge Cases
- Decompression bomb: Warning correctly fires for images > 16M pixels (verified in test output).
- Animation rejection: Multi-frame images are rejected as noncompliant.
- MIME/magic byte mismatch: Both layers of validation must agree.
- Truncated files: Detected and rejected.
- Unauthorized access: Returns 403 for non-owner/non-admin users.
- URL and path injection: Blocked, no network requests made.
- Legacy `creator_avatar_url`: Existing URLs are reported without fetching; new writes are rejected.
