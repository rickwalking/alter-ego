## Risks

| Risk | Severity | Mitigation |
|---|---|---|
| `sanitize_llm_input()` destroys web content | **HIGH** | Use `sanitize_web_content()` — separate function that strips only HTML tags + injection patterns, no lowercasing, no paren-stripping |
| Non-dict sources miss `source_type` in `start_from_subagent()` | Low | Pre-existing bug, silent fallback to `"document"` |
| Playwright crashes in serverless/Docker | Low | `headless=True`, 15s timeout, exception caught by graceful degradation |
| Scraped content exceeds 10K limit | Low | Already truncated in `scrape_url()` |
| Container `rag_agent` provider is dead code | None | Not used by any caller — modifying `build_rag_agent()` is correct |
