## What was built

Deterministic web-research enrichment at workflow start + the AE-0249 orphan fix:

1. **`agents/research_enrichment.py`** — `enrich_sources(sources, params)`:
   - Navigates sources with `source_type == "url"` OR bare-URL content (same
     superset rule as the legacy RAG edge) via the shared `ResearchTool`
     (Playwright), guarded by `is_safe_research_url` (SSRF), capped at 5 URLs,
     Semaphore(2) concurrency, `sanitize_web_content` on scraped text, and
     graceful degradation on block/timeout/error (original content kept,
     structured warnings: research_url_blocked / research_url_scrape_failed /
     research_url_cap_hit).
   - DuckDuckGo topic search (`search_web`), top 3 hits appended as
     `web_search` sources (snippet as content, no scraping of hits);
     search failure is non-fatal (research_search_failed).
   - `research_tool=None` → passthrough (CI, tests, kill switch).
2. **Single choke point** — `EditorialWorkflowService.start_workflow` calls
   `orchestrator.enrich_research_sources(...)` inside the research trace, feeds
   enriched sources to BOTH synthesis and the initial brief (outline sees them).
   The duplicated route-edge `_scrape_url_sources` in `api/dependencies/agents.py`
   (which had NO SSRF guard) is deleted; all three entry points (HTTP route, RAG
   chat tool, DeepAgents subagent) now enrich via the service.
   Layering: params dataclass lives in `domain/protocols/carousel.py` (a `domain/models/` home created a domain import cycle, caught by the imports gate),
   enrichment in `agents/` — zero new application→agents or api→infrastructure
   import edges (ratchet-safe).
3. **Orphan fix (AE-0249 completed)** — `RAGAgent._build_subagents()` registers
   `build_researcher_subagent` (scrape_url + search_web + search_documents)
   when carousel tool access + research tool are present.
4. **Kill switch** — `Settings.research_enrichment_enabled` (default true);
   both build sites (`build_editorial_workflow_service`, `build_rag_agent`) pass
   `research_tool=None` when disabled → byte-identical legacy behavior.
