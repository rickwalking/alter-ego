## Goal

Rewrite the Alter-Ego system prompt (v1 → v3) to:
1. Speak informally like Pedro Marins (not a robotic "helpful AI assistant")
2. Never reference documents, knowledge bases, or indexed content in responses
3. NEVER mention company names or list projects by employer
4. Add prompt injection guardrails that feel natural (not security-robotic)
5. Prevent system prompt leakage, jailbreaks, and tool name exposure

## Problem

The v1/v2 prompts caused the agent to respond robotically:
- "The knowledge base doesn't contain enough detail..."
- "According to the documents indexed so far..."
- "I won't make up information that isn't in the knowledge base"
- Listing companies by name like a resume

Additionally, the prompt had weak anti-jailbreak defenses. User messages reach the LLM unsanitized, and the system prompt lacked explicit boundaries against instruction override attempts.

## Scope

- Create `agents/prompts/alter_ego/v2/system.md`
- Update `agents/alter_ego_agent.py` to load v2
- Deploy to production and verify chat responses

## Non-Goals

- Input sanitization in the chat hot path (T1 follow-up: sanitize user messages before LLM)
- History re-sanitization (T1 follow-up)

- Retrieval chunk sanitization (T2 follow-up)
- ADR-009 write-up (post-validation)
