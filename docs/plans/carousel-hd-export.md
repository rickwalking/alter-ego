## Architecture

```
Pipeline export тЖТ pt/slide_N.jpg (1080x1350)
                тЖТ pt/hd/slide_N.jpg (2160x2700)

Preview endpoint тЖТ tries pt/hd/ first
                 тЖТ falls back to pt/
                 тЖТ falls back to images/
                 тЖТ returns 404 if none exist
```
